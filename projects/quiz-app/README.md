# Quiz App

Cartella preparata per una futura mini web app in JavaScript.

Quando il progetto sara pronto:

1. aggiungi `index.html` in questa cartella;
2. aggiungi la preview in `assets/projects/quiz-app-preview.png`;
3. copia il template `futureProjectTemplate` da `data/projects.json` dentro `projects`;
4. imposta `published` su `true`;
5. aggiorna, se serve, il fallback in `assets/js/projects.js`.
